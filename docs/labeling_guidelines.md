# Labeling Guidelines

Classes: person (ID: 0), helmet (1), vest (2)

Labeling rules: cover partially visible helmets, helmets on top of head only, reflective vests covering torso area, exclude mannequins/ads.

Minimum bbox margin: include entire object with a small margin.

Edge cases: small, blurred, or occluded objects must still be labeled if > ~15×15 px at 640×640.
